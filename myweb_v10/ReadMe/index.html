Contents in Source Pack Folder

After purchasing the item and extracting, you will find the following folders:

SourceCodeProject: This folder contains source code, libraries projects which you have to import into Eclipse, do changes and build the application.

ReadMe: This folder contains index.html file, open it and you can get help from this guide.

Getting Started

Before starting to use this source code and developing app you need to first install the following softwares to your computer. If you have already installed these softwares you can skip this step.

1. JDK 5 or above: Install Java Development Kit (JDK) 5 or greater. You can download here.

2. Eclipse IDE and Android SDK:  I prefer you download ADT Bundle because it comes with Eclipse and Android SDK in it. We need 4.0 + or 5.0 Android version to buildthe projects and libraries. OR You can choose Eclipse Indigo or other packages. I prefer Indigo. You can download here. Extract the zip file into a folder for example E:\ANDROID\Eclipse after downloading. Run eclipse.exe. On first time you will be asked for workspace directory. It is a directory where you want your android project to be stored. For now just create new folder Workspace in E:\ANDROID or any other directory as you wish.

3. Android SDK: Install android sdk. You can download here. Now you need to connect Eclipse and Android sdk. On Eclipse GUI, select Windows > Preferences menu on top. Select Android and then locate your SDK Location, on my machine it is G:\EclipseAndSDK\AndroidSDK4.0, and then click Apply.

Importing Project into Eclipse

1. Extract the downloaded archive that you have purchased.

2. Open Eclipse > Select File > Import.

3. In the Import window page select Existing Android Code into Workspace under Android and click Next. In the next window choose Select root directory. Press the Browse button and choose the folder which you need like for AdMob or StartApp from SourceCodeProject folder present in the Project folder. Press Enter and Select/Check all the projects comes in dialog. Check/Select "Copy Projects Into Workspace" Don't forget to do this check otherwise the projects will not linked with required libraries and then Click Finish.

4. Your project will be visible on the Package Explorer. At the left side you can see the project folder. You can not edit required files from main project
Change the constants in AppConfig.java file for your ads ids, settings, email, etc, check all variables for your required settings as shown in below screenshot.

Changing Package Name

You need to change package name in the .java files (src folder), .xml(res-layout) and AndroidManifest.xml.

Replace in java files, res-layout .xml files and AndroidManifest.xml: Open all the files in the src folder, res-layout folder and AndroidManifest.xml at once in Notepad++. Select Find/Replace option and replace from "design.website.template" to "your.package.name". 

Do not select "Match case".

You may find it little hard to replace it in first try if you have not used Notepad++ before, but in few attempts you can do it. For experienced users this is a simple task. Or you can contact me for this if you fall in any problem.

Changing App Name

App Name: Go to res/values/ directory and edit strings.xml file.

To change App name: Enter the name in the below string.

<string name="app_name">My Web App</string>

Changing Application Icon

The application icon is present at res/drawable-{dpi}/ic_launcher.png. Create your icon image with the same file name and replace the existing file.

This takes different sizes icons for different drawable folders:
drawable-xxhdpi 144 * 144
drawable-xhdpi 96 * 96
drawable-hdpi 72 * 72
drawable-mdpi 48 * 48
drawable-ldpi 36 * 36

Refresh the project for changes to take effect, clean and Run.

Enabling AdMob Ads

App comes with AdMob banner ads and Interstitial ads enabled. You can easily enable/disable the ads (true - enabled, false - disabled) in design.website.template.AppConfig.java file, in place of below line value:


AdMob Ads Enabling/Disabling
public static final boolean ADS_ENABLED = true;
public static final boolean INTERSTITIAL_ADS_ENABLED = true;
public static final String BANNER_ID = "YOUR_ADMOB_BANNER_ID";
public static final String INTERSTITIAL_ADS_ID = "YOUR_ADMOB_BANNER/INTERSTITIAL_ID";

Customization & Skins

Theme
You can easily customize this app skins colors to your liking colors. You can change colors in res/values/colors.xml file, in place of below lines color values:

<color name="red">#bf360c</color>
<color name="material_blue_grey_800">#ff37474f</color>
<color name="material_deep_teal_500">#ff009688</color>
<color name="blue">#304f9f</color>

You can make the default skin to any of above as per your choice color:
You need to just update the order of the color. 0(zero) means default color skin. You can find this variable in AppConfig.java file itself:
public static final int DEFAULT_THEME_COLOR = 0;

 public static final int DEFAULT_COLOR = 0;
public static final int RED = 1;
public static final int BLUE = 2;
public static final int GREY = 3;

Splash
Splash Duration : You can update splash duration in AppConfig.java file, in place of below line duration:
public static final int SPLASH_DURATION = 2000; (The duration is in milliseconds so it comes to 2 seconds.)

Splash Screen Logo: You can change the splash screen logo by simply replacing with your image in place of res/drawable-hdpi/splash_image.png
Or you can design this as per your thought.

Website Url
For creating your website app, you need to provide website url, Contact Us page url and About page url in design.website.template.config.AppConfig.java file, in place of below lines url values:

public static final String WEB_SITE_URL = "http://html5games.com/";
public static final String CONTACT_US_URL = "http://www.google.co.in";
public static final String ABOUT_URL = "http://www.google.co.in";

Call and Email
You can easily change phone number for call and email address to contact you from app, in design.website.template.config.AppConfig.java file, in place of below lines values:

public static final String CONTACT_PHONE_NUMBER = "1234567890";
public static final String CONTACT_EMAIL = "testemail@gmail.com";

Settings
You can easily change settings options default values (true - enabled, false - disabled), in design.website.template.config.AppConfig.java file, in place of below lines values:

public static final boolean DEFAULT_CACHE_ENABLED = true;
public static final boolean DEFAULT_JAVASCRIPT_ENABLED = true;
public static final boolean DEFAULT_SCROLLBARS_ENABLED = true;
public static final boolean DEFAULT_ZOOM_CONTROL_ENABLED = true;
public static final boolean BOTTOM_BAR_ENABLED = true;
public static final boolean SLIDING_MENU_ENABLED = true;

Compiling the Application

After customizing this app, you need to compile into an .apk file that can be installed in Android devices. Here are the steps of compiling this application.

1. Select the MyWebApp project in Package Explorer, Right click and select Export.

2. In Export window select Android > Export Android Application. Then, click Next button.

3. Your project name will appear in Project: box. If not, click Browse and find. Click Next button to proceed.

4. Select your keystore or create a new one. In Location, click Browse button to decide keystore name and the location of it. Give a strong password and proceed to next step.

5. Fill the key creation form and press Next.

6. Finally, click Browse button to specify the location where you want to have the generated apk, and then click Finish. Now your application is ready. Install it. Have fun.
